My Projects

Indian States

Indian States Project Description: This project is a dynamic and informative guide to the states of India, built using HTML and CSS. The interactive design allows users to click on any state, which then opens a dedicated page providing detailed information about that state. Each state's page includes key facts such as the capital, population, languages spoken, and cultural highlights, offering a comprehensive look at India's diverse regions. With a focus on simplicity and user experience, the project is an excellent resource for anyone looking to explore the geography and culture of India in an engaging way. The clean design and intuitive navigation make it easy for users to learn more about each state with just a click.
